# 专注计时小站

一个简洁优雅的时间专注工具，支持多用户各自累计专注时长。

输入用户名和任务，设定时长，进入专注状态。完成后自动记录，可查看历史累计统计。

无需安装、无需后端：打开网页即可使用。
可用 GitHub Pages 一键发布，分享给团队或家人。

---

## 功能特点

- 多用户支持 - 每个人拥有独立的专注记录
- 任务历史累计 - 按任务汇总统计时长
- 专注倒计时 - 带视频背景的沉浸式体验
- 暂停/继续 - 灵活控制专注节奏
- 休息模式 - 一键进入5分钟休息
- 本地存储 - 数据保存在浏览器中，隐私安全

## 使用说明

### 开始专注
1. **输入用户名** - 例如：谢老师 / 小轩 / 大轩（必填）
2. **输入任务** - 例如：阅读 / 备课 / 写作
3. **设定时长** - 1-180 分钟，默认 25 分钟
4. 点击「开始专注」

### 专注中
- 左上角显示剩余时间
- 可点击「暂停休息」暂时停止
- 可点击「结束任务」提前结束
- 右侧面板显示历史累计

### 完成后
- 显示本次专注时长
- 可选择：
  - 「开始下个任务」- 继续专注
  - 「进入休息」- 自动设定5分钟休息
  - 「任务结束」- 返回首页

## 替换素材

只需要保证文件名不变，直接替换同名文件即可：

- 替换开始页背景：用你的图片覆盖 `bg-start.webp`
- 替换结束页背景：用你的图片覆盖 `bg-end.webp`
- 替换视频背景：用你的循环视频覆盖 `bg-focus.mp4`

**注意**：GitHub 区分大小写，文件名必须完全一致。

## 数据存储

历史记录保存在浏览器的 `localStorage` 中：

- 每个用户名拥有独立的记录空间
- 不同用户名的数据互不干扰
- 按任务名称自动汇总累计时长
- 清除浏览器数据或换设备会重新开始累计

## 发布到 GitHub Pages

1. 仓库 Settings -> Pages
2. Source 选 "Deploy from a branch"
3. Branch 选 `main`，Folder 选 `/ (root)`
4. 保存后等待生成链接，例如：
   `https://你的用户名.github.io/仓库名/`
