# Focus Timer｜专注计时小站

一个纯静态网页：输入任务与时长 → 进入专注倒计时（走路视频背景）→ 完成后生成记录，并在左上角显示「历史累计」（按任务汇总分钟数）。

无需安装、无需后端：打开网页即可使用。  
可用 GitHub Pages 一键发布，分享给任何人。

---

## 在线使用
如果你是从 GitHub Pages 访问：  
直接打开页面链接即可开始使用。

---

## 本地使用
1. 下载本仓库（Code → Download ZIP）并解压  
2. 双击打开 `index.html`（建议用 Chrome/Edge）

---

## 文件说明
- `index.html`：主页面与全部逻辑
- `bg-start.png`：开始页背景图
- `bg-end.png`：结束页背景图
- `bg-focus.mp4`：专注页走路循环视频背景（建议 1–5MB 以内更流畅）
- （可选）`bg-focus.webm`：视频备份格式，兼容性更好

---

## 替换素材（最常用）
只需要保证文件名不变，直接替换同名文件即可：
- 替换开始页背景：用你的图片覆盖 `bg-start.png`
- 替换结束页背景：用你的图片覆盖 `bg-end.png`
- 替换走路视频：用你的循环视频覆盖 `bg-focus.mp4`

注意：GitHub 区分大小写，文件名必须完全一致。

---

## 历史累计如何保存？
历史记录保存在浏览器的 `localStorage` 中：
- 每个人看到的是自己的累计
- 清除浏览器数据或换设备会重新开始累计

---

## 发布到 GitHub Pages（给别人一打开就能用）
1. 仓库 Settings → Pages  
2. Source 选 “Deploy from a branch”  
3. Branch 选 `main`，Folder 选 `/ (root)`  
4. 保存后等待生成链接，例如：  
   `https://你的用户名.github.io/仓库名/`
